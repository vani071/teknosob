<a name="upgrade-3.0.0"></a>
## Upgrading To 3.0.0

Plugin requires October build 420+ with Laravel 5.5 and PHP >=7.0.

Session lifetime setting was changed from minutes to seconds. Plugin settings will be reset to defaults after update.
